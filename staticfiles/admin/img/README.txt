    Authentication (Autentikasi)
        Fitur pendaftaran (register) untuk pengguna.
        Fitur login untuk pengguna.
        Fitur logout untuk pengguna.
        Proteksi akses terhadap halaman tertentu untuk pengguna yang belum masuk (unauthenticated user).

    Profil Pengguna
        Pengguna dapat mengunggah foto profil.
        Pengguna dapat memperbarui informasi profil mereka (nama, email, dll.).
        Profil pengguna terkait dengan pengguna terkait.

    Artikel (Post)
        Pengguna dapat membuat artikel baru.
        Pengguna dapat melihat daftar artikel.
        Pengguna dapat melihat detail artikel.
        Pengguna dapat memperbarui artikel yang ada.
        Pengguna dapat menghapus artikel yang ada.
        Pengguna dapat menambahkan komentar pada artikel.

    Kategori Artikel
        Artikel dapat dikategorikan dalam beberapa kategori.
        Pengguna dapat melihat daftar artikel dalam kategori tertentu.

    Komentar
        Pengguna dapat menambahkan komentar pada artikel.
        Komentar memiliki informasi tentang penulis, waktu pembuatan, dan teks komentar itu sendiri.

    Tampilan Halaman
        Desain tampilan menggunakan Bootstrap untuk tampilan yang responsif dan menarik.
        Penggunaan tag-teknologi terintegrasi seperti HTML, CSS, dan JavaScript.

    Pengaturan Static dan Media
        Pengaturan untuk mengelola file static (CSS, JavaScript, dll.).
        Pengaturan untuk mengelola file media (gambar, foto profil, dll.).

    Admin Panel
        Akses ke admin panel Django untuk mengelola data pengguna, artikel, kategori, dan lainnya.

    Routing URL (URL Routing)
        Pengaturan URL untuk mengarahkan permintaan pengguna ke tampilan (views) yang sesuai.

    Migrasi Database
        Penggunaan migrasi database Django untuk memperbarui struktur database.

    Pustaka Pihak Ketiga
        Penggunaan pustaka-pustaka pihak ketiga seperti Pillow untuk mengelola gambar.
